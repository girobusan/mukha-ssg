# Site config

`site_dir/config/site.yaml`

```yaml
# site properties
title: "Example site"
motto: "Proud to be default"
author: "Incognito"
# mostly for RSS
image: ""
description: "Some long description, which will be
  visible in RSS feed and..."
url: "https://example.site"
theme: false

# default maximum length of the list;
# max number of entries on single page
# of multipage lists
list_length: 20

#tags...
tags_dir: "/tags"
tags_page: "/tags/index.html"

#RSS props
rss_uri: "/rss.xml"
atom_uri: "/atom.xml"
#count of rss feed items
feed_length: 30
```
